# RAG Chatbot (Project Skeleton)

This repository contains the Phase 1 skeleton for a Retrieval-Augmented Generation (RAG) chatbot.

## Stack
- Backend: Python + Flask
- Vector DB: ChromaDB
- Embeddings: OpenAI text-embedding-3-small
- LLM: OpenAI GPT-4o-mini
- Frontend: HTML/CSS/JavaScript
- Deployment: Render.com (render.yaml included)

## Setup
1. Create a `.env` from `backend/.env.example`.
2. Install dependencies: `pip install -r backend/requirements.txt`.
3. Run locally: `python backend/app.py`.

## Status
